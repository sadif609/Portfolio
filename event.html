NextGen Events Platform

Overview: The NextGen Events Platform is a dynamic, full-stack web application designed to simplify and modernize event management. It features secure user registration and login with role-based access control, ensuring a safe and personalized experience for both organizers and participants. Users can easily create, update, delete, and manage events, including image uploads for a more engaging interface. The platform also supports a comprehensive event registration system, allowing users to join or leave events and view real-time attendee lists. With its responsive, mobile-friendly UI, modern dark theme, and seamless navigation, it offers a smooth and visually appealing experience across all devices. The system includes real-time event tracking and participant management, along with robust API integration, complete with proper error handling and data validation. Finally, the application is cloud-deployed with a scalable database and secure environment configuration, ensuring reliability and performance under varying workloads.

Technologies Used

Frontend: React 18, JavaScript ES6+, CSS3, React Router, Axios, Vite
Backend: Node.js, Express.js, MongoDB, Mongoose, JWT Authentication
Security: Bcrypt.js, CORS, Passport.js, Google OAuth2
Tools: Git, GitHub, Render.com, MongoDB Atlas, Multer (file uploads)

Key Features Implemented

  • User Authentication System : Registration, login, JWT tokens, password hashing
  • Complete CRUD Operations : Create, read, update, delete events with image uploads
  • Responsive Design : Mobile-first approach with 3 breakpoints (480px, 768px, 1024px+)
  • Event Registration System : Users can join/leave events with attendee tracking
  • Modern UI/UX : Dark theme, smooth animations, professional design
  • RESTful API : 20+ endpoints with proper error handling and validation
  • Database Design : MongoDB with 3 collections (users, events, comments)
  • Cloud Deployment : Live application on Render.com with MongoDB Atlas
